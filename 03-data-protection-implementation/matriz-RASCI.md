# Matriz RASCI – Gobernanza del Programa de Protección de Datos

La presente Matriz RASCI define las responsabilidades a nivel de gobernanza del Programa de Protección de Datos Personales, estableciendo los roles que deben participar en cada actividad clave del programa.

**Legenda:**
- **R** = Responsable operativo (ejecuta)
- **A** = Accountable (aprueba, responde)
- **S** = Soporte (apoya)
- **C** = Consultado (debe ser consultado)
- **I** = Informado (recibe información)
